# Wireless Sensor Network for Agriculture

## Overview

Wireless Sensor Networks (WSNs) are used in agriculture to monitor environmental parameters such as soil moisture, temperature, and humidity in real time. This project demonstrates a smart agriculture monitoring system that helps improve irrigation efficiency and crop management.

## Objectives

* Monitor soil moisture levels
* Measure temperature and humidity
* Display real-time readings
* Support smart irrigation systems
* Promote efficient resource utilization

## Hardware Components

* Arduino Uno
* Soil Moisture Sensor
* DHT11 Sensor
* LCD Display
* Breadboard
* Jumper Wires
* Power Supply


## Results

The system successfully monitors soil conditions and environmental parameters, displaying real-time information for smart farming applications.
